# Sena Repo - GalGame 私有库管理器

## 简介

**Sena Repo** 是一款面向多平台的视觉小说私有库管理工具，适合管理部署在远程服务器（如 NAS、树莓派）上的游戏，让使用者能方便地浏览、搜索、下载和安装自己的游戏收藏。

服务端（Docker / Python）负责扫描目录、清洗文件名、刮削元数据；客户端（Windows / Android / Linux）通过 HTTP 连接服务端，提供一体化的游戏库浏览和元数据编辑体验。

## 导入及清洗逻辑

### 目录结构

```
根目录/
  ├── 会社A/                    ← 第一级：会社文件夹
  │   ├── 游戏1/                ← 第二级：游戏文件夹
  │   │   ├── [PC]游戏1.rar
  │   │   └── [Ty]游戏1.zip
  │   ├── 游戏2/
  │   │   ├── [PC]游戏2.zip
  │   │   └── [KRKR]游戏2.zip
  │   └── 游戏3/
  │       └── 直装_游戏3.apk
  └── 会社B/
      └── 游戏4/
          └── 游戏4安卓直装版.apk
```

### 处理流程

**扫描 → 清洗 → 导入 → 刮削**

1. **第一级** → 识别为**会社**，自动作为标签附加到该目录下所有游戏，同时填入**开发商**字段
2. **第二级** → 识别为**游戏项目**
3. **第三级** → 识别为游戏对应的**压缩包文件**，同一游戏文件夹内可包含多个不同平台的版本

### 清洗规则

从压缩包文件名中提取平台类型：

| 文件名 | 游戏名 | 平台 |
|-------|-------|------|
| `[PC]游戏1.rar` | 游戏1 | PC |
| `[KRKR]游戏2.zip` | 游戏2 | KRKR |
| `[ONS]游戏3.7z` | 游戏3 | ONS |
| `[Ty]游戏4.zip` | 游戏4 | Tyranor |
| `直装_游戏5.apk` | 游戏5 | 安卓直装 |
| `Kanon安卓直装版1.0.3.apk` | Kanon | 直装 |

支持的平台标识：`PC`、`KRKR`、`Ty`（Tyranor）、`ONS`、`直装`，末尾为 `.apk` 或文件名含"安卓""直装"字样的自动归类为直装版。

### 导入规则

- 会社文件夹名自动填入游戏的**开发商**字段（已有游戏只在为空时补填，不覆盖手动修改）
- 同一游戏下的不同平台压缩包各生成独立版本条目
- 非压缩包文件（图片、文档等）自动过滤
- 会社名自动作为标签添加

### 刮削

用游戏名 + 会社标签去刮削源匹配元数据。所有刮削源均免认证：

| 刮削源 | 说明 |
|--------|------|
| VNDB Kana v2 | 免认证，中文标题，含游戏时长 |
| Bangumi | 免认证 |
| Steam | 免认证 |
| DLsite | 免认证（建议配日本代理） |
| 月幕 GalGame | 免认证 |
| IGDB | 可选启用，需 Client ID / Secret |

## 主要功能

- **游戏库浏览**
  - 网格 / 列表双视图，封面大小自定义
  - 搜索：按游戏名称、标签快速检索
  - 按会社名、名称、导入时间排序
  - 显示游戏时长（VNDB 数据）

- **元数据编辑**
  - **下载元数据**：选择刮削源 → 搜索 → 对比 → 逐字段勾选应用
  - 封面实时更新，封面上传即时生效
  - 大图上传播放器展示
  - 支持手动编辑所有字段

- **用户系统**
  - 登录 / 注册，注册需管理员审批
  - 首个注册用户自动激活为管理员
  - 管理员可在设置中审批 / 拒绝注册申请
  - 头像上传，个人信息修改
  - 通知系统

- **游戏下载**
  - 选择平台版本一键下载
  - 支持暂停 / 取消 / 断点续传
  - Android 支持解压到外部存储

- **根目录管理**
  - 添加 / 删除多个游戏根目录
  - 一键扫描刷新，支持批量刮削
  - 目录结构选择（会社/游戏 / 仅游戏 / 扁平）

- **初始化向导**
  - 首次连接自动引导创建管理员 + 配置游戏目录 + Steam 补丁
  - 自动扫描补丁索引

- **Steam 补丁注入**（PC 专属）
  - 双 Tab 界面：客户端（匹配本地 Steam 游戏）/ 服务端（管理补丁索引）
  - 自动扫描 steamapps 目录，按 Steam App ID 匹配补丁
  - 关键词快捷匹配自动识别补丁类型（汉化/音声/剧情/额外）
  - 支持自定义关键词配置
  - 编辑补丁参数实时写回服务端
  - 注入进度条，暂停 / 取消支持

- **Windows 安装包**
  - 便携版 zip + 安装包 exe 双通道
  - 安装包含开始菜单快捷方式 + 卸载入口
  - CI 自动编译发布

- **其他**
  - 游戏详情页卡片化布局，封面阴影，平台彩色标签
  - 双击 exe 拉起已运行实例
  - 关闭时最小化到托盘（可选）
  - 个性化主题色
  - 日志系统（7 天自动清理）

## 部署与安装

### 环境要求

- **服务端**：Docker 20.10+（推荐），或 Python 3.11+ 直接部署
- **客户端**：Windows 10+ / Android 8.0+ / Linux（x86_64 / ARM64）

### 服务端部署

**方式一：Docker 命令行**

```bash
docker run -d \
  --name sena-repo \
  -p 11451:11451 \
  -v /path/to/your/games:/games \
  -v /path/to/data:/data \
  -v /path/to/steam_patches:/steam_patch \
  sena-repo:latest
```

> 刮削 API Key 通过环境变量传入（全部可选，不填也能正常使用）：

| 环境变量 | 对应刮削源 | 获取地址 |
|---|---|---|
| `SENA_BANGUMI_TOKEN` | Bangumi（提速率） | https://bgm.tv/dev/app |
| `SENA_VNDB_TOKEN` | VNDB（提速率） | — |
| `SENA_IGDB_CLIENT_ID` | IGDB（可选启用） | https://dev.twitch.tv/console/apps |
| `SENA_IGDB_CLIENT_SECRET` | IGDB（可选启用） | 同上 |

**方式二：Docker Compose**

```yaml
services:
  sena-repo:
    image: sena-repo:latest
    container_name: sena-repo
    ports:
      - "11451:11451"
    volumes:
      - /path/to/your/games:/games
      - /path/to/data:/data
      - /path/to/steam_patches:/steam_patch
    restart: unless-stopped
```

**方式三：直接部署**

```bash
git clone <repo-url> && cd sena-repo/server
pip install -r requirements.txt
python main.py --host 0.0.0.0 --port 11451 --games-path /path/to/games --data-path /path/to/data
```

### 客户端安装

从 [Releases](https://github.com/404-GCross/Sena-Repo/releases) 页面下载：

- **Windows**：安装包 `.exe`（含开始菜单和卸载）或便携版 `.zip`
- **Android**：`.apk`，允许未知来源后安装
- **Linux**：`.AppImage`、`.deb`、`.rpm` 或 `.pkg.tar.zst`

首次启动客户端需填入服务端地址，注册或登录后即可使用。

## 使用方法

### 首次设置

连接服务端 → 初始化向导引导创建管理员 → 配置游戏目录 → 配置 Steam 补丁目录 → 自动扫描。

### 浏览游戏库

- 底部导航切换游戏库 / Steam 补丁 / 我的
- 搜索栏按名称实时筛选
- 排序：支持导入时间、名称、会社
- 点击游戏进入详情页

### 编辑元数据

点击编辑进入编辑器 → "下载元数据"选择刮削源 → 搜索 → 逐字段对比勾选 → 应用。

### 下载游戏

详情页选择版本 → 下载到本地。支持暂停、断点续传。

### Steam 补丁

打开 Steam 补丁页 → 选择 steamapps 目录 → 自动扫描匹配 → 有补丁的游戏点击"注入"。
服务端 Tab 可管理补丁索引、编辑参数、自定义类型关键词。
